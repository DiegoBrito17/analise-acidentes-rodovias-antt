🛣️ Análise de Acidentes em Rodovias Brasileiras (ANTT)



🚦 Dashboard completo baseado em dados reais da ANTT.



🔗 Fonte oficial: https://dados.antt.gov.br/dataset/acidentes-rodovias



O objetivo deste projeto foi transformar uma base extensa e bruta em informação clara, visual e acionável, permitindo entender o cenário de acidentes em rodovias federais ao longo dos anos.



🔍 Principais entregas



KPIs gerais



Análises temporais completas



Distribuição geográfica



Perfil detalhado dos acidentes



Processos completos de ETL e modelagem



Dashboard em Power BI



📊 Conteúdo



Total de acidentes



Pessoas envolvidas



Feridos, graves, ilesos



Vítimas fatais



Análises por ano, mês, dia da semana



Período do dia (amanhecer, dia, noite etc.)



Tipo de pista



Tipo de acidente



Condição climática



Causas principais



🧩 Processos utilizados



Power Query (ETL)



Modelagem de dados



Medidas DAX



Layout personalizado



Mapas, barras, linhas e KPIs
